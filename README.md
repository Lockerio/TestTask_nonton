# TestTask_nonton
___
### Описание

Тестовое задание для НОНТОН.РФ.
Проект представляет собой консольное приложение, на C#.

___
### Функционал

#### Модель
Product:
* id - уникальная строка, которую задает сам пользователь;
* name - строка, название продукта.

#### Функции

* AddProduct - Добавление нового продукта;
* DeleteProduct - Удаление продукта;
* GetName - Получить название продукта по id;
* FindByName - Получить список продуктов по имени.
